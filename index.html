Apa Itu Git?

GIT adalah salah satu version control system yang diciptakan oleh Linus Torvalds, GIT mencatat setiap perubahan file di code atau sourcode kita

Manfaat

Dengan Git, kita bisa menyimpan seluruh versi source code, memudahkan kolaborasi dalam membangun sistem, berkontribusi di proyek open source, melacak setiap perubahan dengan mudah, serta memahami proses deployment modern secara lebih efisien.

Selain manfaat utama, Git juga memiliki banyak keunggulan lain seperti mendukung eksperimen dengan branch, integrasi dengan tool otomatisasi, dan cocok untuk alur kerja tim yang kompleks.

Git's SaaS

Git adalah alat dasarnya, sedangkan GitHub, GitLab, dan Bitbucket adalah layanan berbasis cloud yang memfasilitasi penyimpanan dan kolaborasi proyek Git secara online. Masing-masing memiliki keunggulan tersendiri sesuai kebutuhan tim atau individu.

"Kapan Git Dibutuhkan?"

Git dibutuhkan saat ingin membangun sistem secara kolaboratif, membuat versi dari sistem, melakukan backup source code, serta bekerja sama dengan programmer dari seluruh dunia.
